# Personal Portfolio Website

This website is a two-page personal portfolio website built with semantic HTML and CSS for CMPA 3301 at Texas Tech University; and it's something I plan to keep building on after the course ends!

The site covers who I am, what I work on, and where to find me online. Design-wise, my main inspiration was the website of illustrator and developer Sharlene Yap ([sharyap.com](https://www.sharyap.com/)). It gave me a big push to be a little creative with how the site looks rather than relying on a boring "modern" look :)

You can visit the live website [here](https://beangusu.github.io/Final-Project)!
